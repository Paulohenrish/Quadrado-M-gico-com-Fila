# Quadrado-Magico-com-Fila
A construção ocorre por meio de uma Busca em Largura (BFS) utilizando uma fila para explorar todos os níveis sistematicamente. A poda é aplicada verificando se a soma parcial já excede a constante mágica $M$
Como compilar e executar (Ambiente Linux)

Para rodar este gerador de Quadrado Mágico baseado em Busca em Largura (BFS), certifique-se de ter o compilador `g++` e a ferramenta `make` instalados no seu sistema operacional.

**Passo a passo:**

1. Abra o terminal e vá até a pasta onde os arquivos do projeto (`ex3.cpp` e `Makefile`) estão salvos.
2. Compile o código executando o seguinte comando:
   ```bash
   make
   Após a compilação, o arquivo executável ex3 será gerado. Para executá-lo, digite:
   ./ex3
